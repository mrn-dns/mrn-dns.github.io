Eyeclops - A Phishing Detection Browser Extension

Short Project Description

Eyeclops is a browser extension that uses state-of-the-art large language models to detect phishing attempts in real time. It analyzes various input types such as suspicious URLs, emails, or SMS messages, and gives users a confidence score along with a short reasoning explanation.

Abstract

This project explores the fine-tuning, integration, and deployment of large language models (LLMs) together with the development process of a phishing detection browser extension. Multiple pre-trained large language models are fine-tuned and trained on the specific task of accurately detecting phishing attempts and their performance is compared using various metrics such as accuracy, precision and F1 score. The model that showcases the best performance is chosen to be integrated into the front-end of the browser extension, providing users with a practical tool for identifying potential phishing threats. In addition to the fine-tuned models, an unmodified text generation LLM is used to provide a reasoning behind the output of the classifier model. The use of a fine-tuned model on which time was spent tweaking various parameters in order to achieve best performance and the use of an untouched model emphasizes the ease of integrating general purpose LLMs into applications and aims to encourage adoption of LLMs by other developers in their applications. The extension features an intuitive interface that allows users to input text for analysis, enabling efficient detection of suspicious content.
